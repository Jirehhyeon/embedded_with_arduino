## 📊 학습 진도 관리

### 일일 체크리스트
```cpp
class DailyProgress {
private:
    bool theoryStudy;
    bool practicalWork;
    bool codeCommit;
    bool documentation;
    
public:
    void trackDaily() {
        Serial.println("=== 오늘의 학습 ===");
        Serial.print("이론 학습: ");
        Serial.println(theoryStudy ? "✅" : "❌");
        Serial.print("실습 작업: ");
        Serial.println(practicalWork ? "✅" : "❌");
        Serial.print("코드 커밋: ");
        Serial.println(codeCommit ? "✅" : "❌");
        Serial.print("문서 작성: ");
        Serial.println(documentation ? "✅" : "❌");
        
        calculateScore();
    }
};
```

- [ ] **월요일**: 새로운 개념 학습
- [ ] **화요일**: 실습 프로젝트 시작

- [ ] **목요일**: 기능 확장 및 개선
